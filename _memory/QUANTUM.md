# QUANTUM

## Current Thesis
양자컴퓨팅은 장기 잠재력은 크지만 상용화 시점, 실제 고객 가치, 하드웨어 방식별 승자 예측이 아직 불확실한 초기 테마입니다.

## Management Trust
- 경영진의 과학적 신뢰성뿐 아니라 상업화 로드맵, 고객 PoC 전환율, 자본 조달 방식이 중요합니다.
- 과장된 TAM보다 실제 반복 매출, 정부/기업 계약, 기술 milestone 달성이 더 중요합니다.

## Key Risks
- 기술 방식 간 경쟁: trapped ion, superconducting, photonic, annealing 등
- 상용화까지 긴 시간과 높은 현금 소진
- 잦은 주식 발행과 희석
- 주가가 과학적 진전보다 narrative에 더 민감하게 움직일 위험

## Watchpoints
- 실제 고객 계약과 반복 매출
- error correction, logical qubit, gate fidelity 등 milestone
- 현금 runway와 증자 가능성
- 대형 클라우드/반도체 기업과의 경쟁 또는 파트너십

## Last Emily View
양자컴퓨팅은 portfolio의 optionality로는 흥미롭지만, 개별 기업은 기술 증거와 재무 runway를 매우 엄격하게 봐야 합니다.
